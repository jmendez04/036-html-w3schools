Título de la Página en HTML

El elemento <title> define el título que aparece en la pestaña del navegador.

Título básico

<title>Mi Página Web</title>

Ejemplo

Observa la pestaña del navegador para ver el título de esta página.

Otro ejemplo de título

<title>Curso de HTML</title>

Uso del elemento <title>

  • Muestra el nombre de la página en la pestaña del navegador.
  • Se utiliza cuando la página se agrega a favoritos.
  • Ayuda a los motores de búsqueda (SEO).
