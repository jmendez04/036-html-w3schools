Favicon en HTML

Esta página utiliza un favicon. Observa la pestaña del navegador para verlo.

Favicon con archivo ICO

El favicon puede agregarse utilizando un archivo .ico. 

<link rel="icon"
      type="image/x-icon"
      href="favicon.ico">

Favicon con archivo PNG

<link rel="icon"
      type="image/png"
      href="favicon.png">

Elemento utilizado

El favicon se agrega mediante la etiqueta <link> dentro del elemento <head>.
